Lorenz System

Numerical solution to ODE using RK4

Lorenz system is a classic example of those which behave chaotically. It consists of a coupled system of three differential equations. Runge kutta method of order 4 is a common method for approximation of solutions to equations in the following seeming y'(t)=f(y(t),t).

This system of ODE is an oversimplified version of the Navier-Stokes equations. They emerge from a problem which concerns a fluid in a container whose top and bottom surfaces are held at different temperatures.

Solution to Laplace equation

Numerical solution using relaxation algorithm

In this project, it is given a numerical solution to Laplace equation over a region with nor cartesian neither cylindrical symmetry. The boundary conditions are given over the outer square and an inner circumference of the region of interest.
