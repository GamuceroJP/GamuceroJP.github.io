Lorenz System

Numerical solution to ODE using RK4

Lorenz system is a classic example of those which behave chaotically. It consists of a coupled system of three differential equations. Runge kutta method of order 4 is a common method for approximation of solutions to equations in the following seeming y'(t)=f(y,y(t)).
